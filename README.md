# 🛡️ Suraksha – Women Safety App

A Python-based desktop application designed to enhance women's safety by enabling quick emergency communication and location sharing. The project focuses on providing a simple, reliable, and user-friendly solution that can be used during emergency situations.

---

## 📌 Overview

Suraksha – Women Safety App was developed as a mini project for the **Programming Skills in Python (VSEC02)** course. The application demonstrates how Python can be used to build practical real-world solutions by integrating a graphical user interface with backend functionality.

The primary objective of the project is to provide users with a fast and efficient way to seek help during emergencies through instant alerts and location sharing.

---

## ✨ Features

- 🚨 One-click SOS emergency alert
- 📍 Share current location with emergency contacts
- 🖥️ User-friendly desktop interface
- ⚡ Quick emergency response mechanism
- 🔐 Simple and easy-to-use design

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)

## 🚀 Getting Started

### Prerequisites

- Python 3.x
- pip

### Installation

Clone the repository:

```bash
git clone https://github.com/Harshita-coding/Suraksha_WomenSafetyApp.git
```

Navigate to the project folder:

```bash
cd Suraksha_WomenSafetyApp
```

Install the required dependencies (if applicable):

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
